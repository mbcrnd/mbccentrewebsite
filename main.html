User Role
  • User သတ်မှတ်ရာတွင် Role အလိုက်Level သတ်မှတ်လို့ရပါသည်။
    ဥပမာ ။ ။ ပြင်ခွင့်ဖျက်ခွင့်ရှိသည့် Role, Transaction ပဲရိုက်ခွင့်ရှိသည့် Role အစရှိသဖြင့် မိမိသတ်မှတ်လိုသည့် အတိုင်း Role တွင် သတ်မှတ်ခဲ့ရပါမည်။
  • Role ရှိ Menu Tree အောက်ရှိ menu ၏ ဘေးရှိ Details ကိုနှိပ်လိုက်ပါက

    ပြထားသည့်ပုံအတိုင်းကျလာပါလိမ့်မည်။
  • All ကို Ticking ခြစ်လိုက်ပါက
    Add=အသစ်ဆောက်ခွင့်ှရှိခြင်း၊Edit = ပြန်ပြင်ခွင့်ရှိုခြင်း၊Delet = ဖျက်ခွင့်ရှိခြင်း၊View = Report ကြည့်ခွင့်ရှိခြင်း စသည့် အားလုံးကို အသုံးပြုခွင့်ရှိကြောင်း သတ်မှတ်ပေးလိုက်ခြင်းဖြစ်ပါသည်။
  • အကယ်၍ ပြင်ခွင့်၊ဖျက်ခွင့် မပေးချင်ပါက Edit နှင့် Delete ကို Ticking ဖြုတ်ခဲ့ပေးရပါမည်။
  • ပေးချင်သော menus များ Ticking ခြစ်ပေးပြီးပါက save button ကိုနှိပ်ပေးရပါမည်။
User Create

Administion > User > Users တွင် အသစ်တည်ဆောက်လိုပါက New ကို နှိပ်ပါ။

  • Login In ID နေရာတွင် Code No ရိုက်ထည့်ပေးပါ။
  • User Name နေရာတွင် Name ကို ဖြည့်ပါ။
  • Password နေရာတွင် မိမိဖြည့်သွင်းလိုသော password ကို ရိုက်ထည့်ပေးပါ။
  • Confirm Password နေရာတွင် အပေါ်တွင်ဖြည့်သွင်းထားသော password အတိုင်း မှန်ကန်စွာ ရိုက်ထည့်ပေးပါ။
  • Role တွင် User နှင့် သက်ဆိုင်သော role ကို ချိတ်ပေးပါ။
  • Allowance Edit Day ဆိုသည်မှာ မိမိသတ်မှက်ခဲ့သော User ကို ရက်အလိုက် ပြင်ခွင့်သတ်မှတ်ပေးခြင်း ဖြစ်ပါသည်။ ဥပမာ တစ်ရက်အတွင်းပြင်ခွင့် ပေးမည်ဆိုလျှင် (1) နှစ်ရက်အတွင်း ပြင်ခွင့်ပေးမည်ဆိုလျှင် (2)စသည်ဖြင့် သတ်မှတ်ပေးခြင်းဖြစ်ပါသည်။
  • POS အရောင်းရိုက်မည့် user ဖြစ်မည်ဆိုပါက Cashier ကို ‘On” ခဲ့ပေးရန် လိုအပ်ပါသည်။
  • User အသုံးပြုမည့် Branch ကို Ticking On ပါ။
  • ထည့်သွင်းထားသော users များကို ပြန်လည်ကြည့်ရန်အတွက် List button ကို နှိပ်၍ Preview တွင် ကြည့်နိုင်ပါသည်။
